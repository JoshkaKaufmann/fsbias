# fsbias
data repositery for gender/career biases in FSBI symposiums

# final files

#1

FSBI_IFM_timing_2025.csv includes timing measured for a number of presenters during FSBI-IFM 2025 symposia.
first_name: first name
gender: assessed gender
career: assessed career stage
tot_sec : total number of seconds taken

#2

time_series.csv includes data for analysis of temporal gender representation in FSBI symposium.
first_name : first name of presenter
year_symposium : year the symposium took place
location_symposium : city/University where the symposium took place	
final_gender : final gender
invited : if presenter was invited or not (includes James Jones lectures and plenaries)
